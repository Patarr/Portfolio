Maquettebouw

Mijn vorige opleiding nam plaats in een pand met historische waarde. In mijn laatste jaar daar heb ik als schoolproject een schaalmodel ervan gemaakt.

Hoe ben ik te werk gegaan?

Via mijn decaan had ik de bouwtekeningen ontvangen die ik kon laten inscannen. Maar voordat ik echt aan het werk kon ben ik eerst gaan kijken naar hoe ik hem daadwerkelijk ging produceren. Ik had de mogelijkheid ter beschikking om mdf platen via een laser graveer machine uit te snijden. Het enige probleem hiermee was is dat deze een maximaal formaat van circa 30x40cm per keer kan inladen. Ik moest het dus gaan opdelen in stukken. Daarbij was het verstandig om vooraf een schaal te bepalen over hoegroot ik het eindproduct wilde maken. Mijn uiteindelijke doel was om met LEGO poppetjes te werken en ook interieur erbij te bouwen. Mijn schaal heb ik dus op een LEGO poppetje aangepast.

Toen was het eindelijk zover. Nu kon ik via de tekeningen alles gaan vectoriseren. Elk raam, vensterbank en aansluitingsstuk moest aan gedacht worden. Ik moest dus heel secuur aan het werk gaan dat ik geen milimeter afwijking heb. Anders past het dadelijk niet.

Voordat ik de gedetaileerde heb gemaakt heb ik eerst een 'kale' versie gemaakt. Hier kon ik zien hoe het eruit kwam te zien en of alles wel past. Ook kon ik op basis van dit prototype aanpassingen maken.

Na de aanpassingen kwam het leuke werk. Nu kon ik hem aankleden. Via een UV-printer heb ik het bakstenen patroon aan kunnen brengen. De kozijnen en daktegel - die overgens van ijsstokjes gemaakt zijn - heb ik achteraf geverft.
